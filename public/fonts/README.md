# Fuentes del Proyecto

## Spring LP

Para que la fuente Spring LP funcione correctamente, necesitas colocar los archivos de fuente en esta carpeta:

1. `SpringLP.woff2` (recomendado - mejor compresión)
2. `SpringLP.woff` (fallback)
3. `SpringLP.ttf` (fallback adicional)

### Dónde conseguir la fuente:

La fuente Spring LP es una fuente comercial. Puedes:
- Comprarla en sitios como MyFonts, FontSpring, etc.
- Usar una alternativa gratuita similar como:
  - "Pacifico" (Google Fonts)
  - "Dancing Script" (Google Fonts)
  - "Satisfy" (Google Fonts)

### Alternativa temporal:

Si no tienes la fuente Spring LP, el CSS ya está configurado con fuentes fallback:
- 'Brush Script MT' (disponible en muchos sistemas)
- 'cursive' (fuente cursiva genérica del sistema)

### Cómo agregar los archivos:

1. Descarga los archivos de fuente Spring LP
2. Colócalos en esta carpeta (`public/fonts/`)
3. Asegúrate de que los nombres coincidan:
   - SpringLP.woff2
   - SpringLP.woff
   - SpringLP.ttf
4. La fuente se aplicará automáticamente a todos los títulos de sección
