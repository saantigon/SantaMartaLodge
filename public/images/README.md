## Formatos recomendados:
- **Formato**: JPG o WebP
- **Resolución**: 1200x800px mínimo
- **Peso**: Máximo 500KB por imagen
- **Aspect ratio**: 4:3 para galería, 16:9 para hero

## Uso:
1. Coloca las imágenes en las carpetas correspondientes
2. Las imágenes aparecerán automáticamente en la galería
3. Para el hero, las imágenes flotantes se pueden personalizar en `Hero.tsx`
